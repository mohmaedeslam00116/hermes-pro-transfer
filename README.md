## 🛠️ التقنيات المدعومة

### 1. HTTP Server
- الطريقة الأكثر موثوقية
- تعمل على أي شبكة Wi-Fi
- لا تحتاج لأي إعداد

### 2. Wi-Fi Direct
- اتصال مباشر بين الأجهزة
- لا يحتاج راوتر
- سرعات عالية جداً

### 3. WebRTC
- تقنية حديثة P2P
- تشفير تلقائي
- تأخير منخفض
