## 🎯 What's New

### Core Features
- ✅ **Send Files** - Share files to any device on your local network
- ✅ **Receive Files** - Download files shared by other devices
- ✅ **QR Code Transfer** - Quick connection via QR code scanning
- ✅ **Multiple Technologies** - HTTP Server, Wi-Fi Direct, WebRTC support

### UI/UX
- ✅ **Onboarding Flow** - Smooth first-run experience
- ✅ **Dark/Light Theme** - Toggle between themes
- ✅ **Material Design 3** - Modern and clean interface
- ✅ **Progress Tracking** - Real-time transfer progress

### Technical
- ✅ **Provider State Management** - Clean architecture
- ✅ **Android SDK 26+** - Android 8.0 Oreo and above
- ✅ **CI/CD Pipeline** - Automated builds with GitHub Actions

## 📱 Installation

1. Download the APK from this release
2. Enable "Install from unknown sources" in your device settings
3. Open the APK and tap Install
4. Launch Hermes and grant the requested permissions

---

## ⚠️ Known Issues (Beta)

1. **Wi-Fi Direct** - Requires additional platform setup
2. **WebRTC** - Signaling server not included
3. **Large Files** - Files >1GB may cause memory issues

---

## 🔧 Requirements

- Android 8.0+ (API 26+)
- Wi-Fi connection (same network as receiver)
- Camera permission (for QR scanning)
